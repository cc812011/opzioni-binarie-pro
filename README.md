# Previsioni Opzioni Binarie PRO

**Strumento di analisi tecnica** per opzioni binarie basato su Python.

Calcola indicatori affidabili (ADI, RSI, MACD, Bollinger Bands, medie mobili) ed genera segnali semplici di **Call** e **Put**.

### Funzionalità
- Supporto per file CSV o download automatico da Yahoo Finance (yfinance)
- Calcolo sicuro degli indicatori (risolve i problemi comuni con pandas_ta)
- Generazione automatica di segnali di trading
- Grafico interattivo salvato come file HTML
- Facile da usare e da modificare
